# CCI-AB Website

Site institucional da Camara de Comercio e Industria Angola - Botswana.

## Requisitos

- Node.js 20+

## Como rodar o projeto

1. Instalar dependencias:

```bash
npm install
```

2. Iniciar em modo desenvolvimento:

```bash
npm run dev
```

3. Abrir no navegador:

```text
http://localhost:5173
```

## Scripts disponiveis

- Desenvolvimento:

```bash
npm run dev
```

- Gerar build de producao:

```bash
npm run build
```

- Visualizar build localmente:

```bash
npm run preview
```

- Rodar linter:

```bash
npm run lint
```
